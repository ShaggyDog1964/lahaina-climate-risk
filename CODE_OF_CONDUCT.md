# Code of Conduct

## Our Standards

This is an academic research repository. We expect all contributors and participants to:

- Use precise, professional language in all interactions
- Engage substantively with methodological disagreements — explain the identification concern,
  the reference paper, and the proposed fix
- Treat replication failures as scientific problems to be solved, not personal failures
- Distinguish between bugs (code does not implement the stated method) and disagreements
  (the stated method is wrong) — the latter belong in a paper, not an issue

## What We Don't Tolerate

- Personal attacks or harassment of any kind
- Dismissing concerns without engaging with the substance
- Claiming credit for others' contributions
- Violating data use agreements (see `docs/data_acquisition.md`)

## Reporting

If you experience behavior that violates these standards, contact the repository maintainers
via GitHub's private reporting mechanism. All reports will be reviewed.

## Enforcement

Violations will result in warnings, temporary bans, or permanent removal from the project
at the maintainers' discretion.

---

This project follows the spirit of the [Contributor Covenant](https://www.contributor-covenant.org/),
adapted for academic research contexts.
